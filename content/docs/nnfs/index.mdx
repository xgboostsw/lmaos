# Neural Networks from Scratch

This section contains tutorials and exercises for building neural networks from scratch, implementing backpropagation, activation functions, and optimization algorithms without relying on high-level ML frameworks.

## Available Labs

 
- **NNFS Lab 01**: McCulloch-Pitts Neuron - Understanding the earliest computational model of a neuron
- **NNFS Lab 02**: Single Layer Perceptron - Learning AND/OR/NAND/NOR with perceptron learning rule
 - **NNFS Lab 03**: Single Layer Feedforward Network - Activation functions demo (linear, ReLU, sigmoid, bipolar sigmoid)
- **NNFS Lab 04**: Multiple Layer Perceptron - Forward pass and activation examples (Exp4)
- **NNFS Lab 05**: Single Layer Neural Network Training - Tiny training loop (Exp5)
- **NNFS Lab 06**: MultiLayer NN Training - Training and forward-pass example (Exp6)
- **NNFS Lab 07**: Activation Functions and Their Derivatives - Visualize activations and derivatives using TensorFlow (Exp7)
- **NNFS Lab 08**: Multi Layer Perceptron Using TensorFlow For AND Gate - Small MLP trained for the AND gate (Exp8)
